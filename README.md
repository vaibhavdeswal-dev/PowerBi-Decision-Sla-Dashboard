# Decision Efficiency & SLA Performance Dashboard

This project presents a Power BI dashboard focused on analyzing decision efficiency and SLA performance across different request types and priority levels. The dashboard is designed to help identify delay patterns, SLA risks, and process bottlenecks in a decision-based workflow.

---

## Dashboard Overview

### Page 1: Decision Performance Overview
This page provides a high-level view of overall system performance, including:
- Total requests received
- Average decision, review, and finalization time
- SLA breach rate
- Request volume trends over time

It is intended for quick monitoring and operational overview.

![Decision Performance Overview](images/page1_overview.png)

---

### Page 2: Process Delay & Risk Drivers
This page focuses on deeper analysis to understand what drives delays and SLA breaches:
- SLA breach rate by request type
- Decision time by priority level
- Impact of review delays on total decision time
- Decision time vs number of revisions
- Breakdown of delays across process stages

This view helps identify high-risk areas and improvement opportunities.

![Process Delay & Risk Drivers](images/page2_analysis.png)
